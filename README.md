# Agente de IA 24h – Atendimento Automatizado via WhatsApp

## Sobre o Projeto

Agente de Inteligência Artificial desenvolvido para atendimento automatizado 24 horas via WhatsApp.  
O sistema processa mensagens automaticamente, responde com lógica conversacional estruturada e realiza registro inteligente de leads em banco de dados.

O foco do projeto é automação de atendimento com humanização de respostas e organização de dados para otimização de processos comerciais.

---

## Funcionalidades

- Atendimento automatizado 24h
- Processamento inteligente de mensagens
- Buffer para simulação de digitação
- Envio de mensagens fragmentadas para humanização da conversa
- Verificação e registro automático de leads no banco de dados
- Estrutura escalável para integração com APIs externas
